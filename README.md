# DB Hydro Data Fetch Toolkit


## What it does

- Retrieves data from DBHYDRO stations listed in dbhydro.cfg
- Creates Pandas data frames for each station
- Writes data frame out as CSV in reports folder
- Writes logging data in logs folder
- Uses new GitHub Token-based auth
- Rules the universe

This is a work in progress and will change on a daily basis
